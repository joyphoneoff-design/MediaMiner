# MediaMiner 新手手冊 v2.0

> **版本**: 2.0 (2026-01-06)
> **適用對象**: 創業者、知識工作者
> **核心功能**: 社交媒體內容擷取與知識提取

---

## 🌟 v2.0 重大更新

- **Whisper 後端選擇**: 支援 Groq API (免費超快)、MLX (本地 GPU)、OpenAI
- **本體論驗證**: entities/tags 100% 符合預設清單，自動模糊匹配
- **繁體中文 + 台灣用語**: 所有輸出自動轉換為台灣用語
- **單次 API 調用優化**: 知識提取 + 摘要 + 關鍵字 + 實體 + 標籤一次完成
- **逐字稿格式化**: 自動添加標點符號與斷句

## 🚀 快速開始

### 1. 啟動系統
```bash
cd ~/MediaMiner
source .venv/bin/activate
streamlit run ui/app.py --server.port 8502
```

### 2. 開啟 Web UI
瀏覽器開啟: http://localhost:8502

### 3. 基本使用流程
1. 輸入 YouTube 頻道 URL
2. 選擇要下載的影片
3. 選擇 Whisper 後端（建議：Groq API）
4. 自動擷取字幕並提取知識
5. 輸出含完整 YAML frontmatter 的 MD 檔案

---

## 🎙️ Whisper 後端選擇

| 後端 | 速度 | 費用 | 適用場景 |
|------|------|------|---------|
| **Groq API** ⭐ | 超快 | 免費 | 日常使用首選 |
| **MLX** | 快 | 免費 | 無網路時使用 |
| **OpenAI** | 中等 | 付費 | 最高準確度 |

### 並行處理
- **Groq/OpenAI**: 支援多線程並行 (1-10 workers)
- **MLX**: 單線程 (GPU 最佳化)

## 📋 輸出格式

每個影片輸出一個 MD 檔案，包含：

```yaml
---
title: 影片標題
source: youtube
author: 頻道名稱
guest: 訪談嘉賓 (若有)
url: https://...
duration: "15:30"
keywords: [關鍵字1, 關鍵字2, ...]
summary: "核心觀點摘要..."
entities: [商業模式, 內容行銷, ...]  # 100% 符合本體論
tags: [市場定位, 自動化, ...]        # 100% 符合預設集
---

## 逐字稿全文
（含標點符號、繁體中文、台灣用語）

---

## AI 知識提取
（核心觀點、可執行建議、黃金句子等）
```

## 🔑 API 配置

### 優先順序 (免費優先)
1. **Cerebras** - 免費
2. **Gemini 2.5 Flash** - 免費
3. **OpenRouter** - 部分免費
4. **LM Studio** - 本地免費
5. **OpenAI** - 付費備用

### 配置 API 密鑰
```bash
nano ~/MediaMiner/config/api_keys.env
```

必要密鑰：
```bash
GROQ_API_KEY=your_groq_key        # Whisper API
CEREBRAS_API_KEY=your_key         # LLM 首選
GEMINI_API_KEY=your_key           # LLM 備援
```

## ❓ 故障排除

### Q: Whisper 下載模型卡住？
首次使用 MLX 後端會下載 ~1.6GB 模型，請耐心等待。

### Q: entities/tags 不對？
系統會自動進行後處理驗證，將 LLM 輸出映射到最接近的預設項目。

### Q: 簡體中文沒轉換？
確認 LLM 提供商正常運作，系統會在 prompt 中強調使用繁體中文+台灣用語。
