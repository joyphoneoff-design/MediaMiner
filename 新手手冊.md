# Crawl_R2R 新手手冊 v1.0

> **版本**: 1.0 (2025-12-28)
> **適用對象**: 創業者、知識工作者
> **核心功能**: 社交媒體內容擷取與知識提取

---

## 📚 系統概述

Crawl_R2R 是一個專為一人公司創業者設計的社交媒體知識提取框架，支援：
- 📺 YouTube 頻道批次擷取
- 📱 小紅書內容處理
- 🤖 LLM 知識提取
- 🔗 R2R 向量庫整合

---

## 🚀 快速開始

### 1. 啟動系統
```bash
cd ~/Crawl_R2R
./run.sh
```

### 2. 開啟 Web UI
瀏覽器開啟: http://localhost:8502

### 3. 基本使用流程
1. 輸入 YouTube 頻道 URL
2. 選擇要下載的影片
3. 自動擷取字幕並提取知識
4. 同步至 R2R 向量庫

## 🔑 API 配置

### 優先順序 (免費優先)
1. **Gemini 2.5 Flash Lite** - 免費
2. **Cerebras** - 免費
3. **OpenRouter** - 部分免費
4. **LM Studio** - 本地免費
5. **OpenAI** - 付費備用

### 配置 API 密鑰
```bash
# 複製範例文件
cp config/api_keys.env.example config/api_keys.env

# 編輯並填入您的密鑰
nano config/api_keys.env
```

## ❓ 故障排除

### Q: 啟動失敗？
```bash
# 檢查依賴
pip3 install -r requirements.txt

# 重新啟動
./run.sh
```

### Q: YouTube 下載失敗？
```bash
# 更新 yt-dlp
pip3 install --upgrade yt-dlp
```

### Q: API 錯誤？
1. 檢查 `config/api_keys.env` 密鑰是否正確
2. 確認 Gemini 配額未用盡
3. 系統會自動嘗試備援 API
