# MediaMiner 重建手冊 v3.0

本手冊說明如何從零開始重建 MediaMiner 系統。

> **版本**: 3.0 (2026-01-10)

---

## 系統需求

- macOS (Apple Silicon 推薦)
- Python 3.11+
- 約 3GB 磁碟空間

---

## 步驟一：克隆專案

```bash
cd ~
git clone https://github.com/joyphoneoff-design/MediaMiner.git
cd MediaMiner
```

---

## 步驟二：建立虛擬環境

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
pip install mlx-whisper  # Apple Silicon 本地推理
```

---

## 步驟三：配置 API 密鑰

```bash
cp config/api_keys.env.example config/api_keys.env
nano config/api_keys.env
```

必要密鑰：
```bash
GROQ_API_KEY=your_key
CEREBRAS_API_KEY=your_key
```

---

## 步驟四：驗證安裝

```bash
source .venv/bin/activate
streamlit run app.py --server.port 8502
```

開啟 http://localhost:8502

---

## 目錄結構

```
~/MediaMiner/
├── app.py                        # Streamlit UI
├── config/api_keys.env            # API 密鑰
├── scrapers/                      # 爬蟲模組
├── processors/knowledge_extractor.py  # 知識提取
└── requirements.txt
```

## 故障排除

**Q: MLX Whisper 模型下載失敗？**
```bash
huggingface-cli download mlx-community/whisper-large-v3-turbo
```

**Q: 本體論載入失敗？**
```bash
ls ~/R2R/config/ontology/
```
